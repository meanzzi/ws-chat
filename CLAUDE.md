# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 프로젝트

WebSocket 학습용 실시간 1:1 채팅 애플리케이션.

## 기술 스택

- **프론트엔드**: React + Vite + JavaScript + Tailwind CSS
- **백엔드**: Node.js + `ws` 패키지
- **통신**: WebSocket (`ws://localhost:8080`)

## 문서 참조

요구사항, 설계, UI/UX 스펙은 `docs/` 폴더를 확인할 것.

- `docs/prd.md` — 기능 요구사항
- `docs/design.md` — 전체 설계 (폴더 구조, 컴포넌트, 프로토콜, 개발 순서)
- `docs/ux.md` — UI/UX 스펙 (색상, 반응형, 레이아웃)
- `docs/conventions.md` — 개발 컨벤션 (패키지 매니저, 린터, 커밋 규칙)
