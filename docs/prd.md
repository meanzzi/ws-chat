# PRD — 실시간 WebSocket 채팅방

> PRD 작성을 위해 아래 질문들에 답해주세요.

---

## 1. 사용자 및 목적

- 대상 사용자 : 본인 (혼자 공부용)
- 동시에 2명 접속 예정

## 2. 채팅방 구조

- 접속 시 자동으로 채팅방 입장
- 닉네임 입력 필요

## 3. 메시지 기능

- 텍스트만 지원
- 새로 접속 시 비어있음. DB 사용하지 않음. 보존은 추가 예정(DB 구축 완료 시)
- 버튼 클릭 시 전송 가능

## 4. 접속 상태

- 현재 접속자 수나 접속자 목록 필요없음.
- 누군가 입장 퇴장시 시스템 메시지 필요없음. 1:1로 이루어진 채팅방

## 5. UI / 프론트엔드

- React + Vite + JavaScript 사용
- Tailwind CSS 사용
- 반응형 필요

## 6. 백엔드 및 배포

- Node.js 사용
- 현재는 로컬에서 작동

## 7. 범위 조정

- 포함되어야 하는 기능 : 닉네임 입력, 메세지 전송/수신, 텍스트만(DB 없음)
- 없어도 되는 기능 : 접속자 수/목록, 입장/퇴장 시스템 메시지, 여러 채팅방
