# UI/UX 설계 — 질문 및 결정사항

## 1. 색상 테마

- 전체적인 톤은 밝은 화이트 컬러(기본)
- 포인트 컬러 : #bdbbff

## 2. 메시지 버블

- 내 메시지와 상대방 메시지 구분 : 좌(상대) / 우(나) 정렬 + 색상 차이 (카카오톡 스타일)
- 각 메시지에 타임스탬프(시:분) 표시

## 3. 입력창 UX

- 메시지 전송 방법 : 버튼 클릭만
- 빈 메시지(공백)는 전송을 막지않음. 빈메세지도 전송 가능

## 4. 연결 상태 표시

- WebSocket 연결이 끊겼을 때 "연결이 끊겼습니다"라는 배너 표시

## 5. 빈 상태 (Empty State)

- 채팅방 입장 직후 메시지가 없을 때 안내 문구 표시

## 6. 닉네임 입력 화면

- 닉네임 글자 수 제한 최대 10자
- 닉네임 입력 후 Enter 키로도 입장 가능

## 7. 반응형 기준

- 모바일 기준 최소 너비 : iPhone14 기준
- 데스크탑에서 채팅창 최대 너비를 제한 : 중앙 정렬 + 최대 너비 제한
